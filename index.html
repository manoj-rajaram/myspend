4 Steps — One-Time Setup

Takes about 5 minutes. Works only when the app is hosted online (e.g. GitHub Pages) — not from a local file.

1
Create a Google Sheet
Open Google Sheets and create a new blank spreadsheet named MySpend.
➡️ Open Google Sheets
2
Open Apps Script
In your sheet: Extensions → Apps Script. Delete all existing code in the editor.
3
Paste Script & Deploy
Copy the code below → paste into Apps Script → 💾 Save (do NOT click Run ▶️) → Deploy → New Deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy → copy the URL.

💡 Tips
📱 On mobile: Host on GitHub Pages → open the link in Chrome → works like a real app
🚫 Local file won't work: Rename to index.html and upload to GitHub Pages
📊 View raw data: Open your Google Sheet to see all expenses in a spreadsheet
🔄 Auto-sync: App loads latest data from Google Sheets every time it opens
🔒 Privacy: Data goes only to your own Google account
